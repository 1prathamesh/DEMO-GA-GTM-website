📊 GA4 Configuration

Configure your GA4 Measurement ID and tracking settings for the store.

GA4 Measurement ID
Your Measurement ID looks like G-XXXXXXXXXX. In GA4 go to Admin → Data Streams → your stream.
Used to connect GA4 tags in GTM. You'll create a GA4 Configuration Tag in GTM using this ID.
Optional. These will be pushed into the dataLayer as user_properties.
🔗 GTM Tag Setup Guide
Once you have your GA4 ID, create this tag in GTM:
1️⃣
Create a GA4 Configuration Tag
In GTM → Tags → New → Google Analytics: GA4 Configuration. Enter your Measurement ID. Set trigger to All Pages.
2️⃣
Create Event Tags for each dataLayer event
Tag type: GA4 Event. Event name: {{Event}}. Trigger: Custom Event matching each event name (e.g. add_to_cart).
3️⃣
Map dataLayer variables
Create Data Layer Variables in GTM for fields like ecommerce.value, ecommerce.items, etc.
4️⃣
Use GTM Preview to test, then Publish
Open the store with GTM Preview active. Fire events and verify tags fire with correct data before publishing.
